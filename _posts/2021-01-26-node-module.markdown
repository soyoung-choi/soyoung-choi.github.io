Node.js는 기능을 확장할 때 사용하는 개념이 바로 모듈이다. 내부 모듈과 외부 모듈의 차이점에 대해서 알아보자.

&nbsp;

## 내부 모듈

Node.js에  기본적으로 존재하는 모듈을 내부 모듈이라고 불른다. 내부 모듈은 <https://nodejs.org/dist/latest-v6.x/docs/api/>에서 확인할 수 있다. 주소창에 v6-x 부분을 변경하면, 다른 버전의 최신 문서로 확인할 수 있다.

## 외부 모듈

Node.js에 기본적으로 존재하지 않고, 개인 또는 단체가 만들어 배포하는 모듈은 외부 모듈이라고 한다. 외부 모듈은 <https://npmjs.org/> 에서 확인할 수 있다.  

외부 모듈을 사용하려면 별도로 설치해야 한다. 다음 명령어를 통해 외부 모듈을 설치할 수 있다.
`npm install 모듈명`  

설치한 다음에 자바스크립트 파일 내부에서 모듈을 추출하여 사용한다.
`var 모듈명 = require('모듈명');`
